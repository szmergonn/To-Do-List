# To-Do List

Небольшое To-Do приложение на чистом JavaScript.

## Функциональность
- добавление задач
- отметка задачи как выполненной
- удаление задач
- сохранение списка в localStorage

## Как устроена логика
В качестве источника данных используется массив `tasks`.
Каждая задача - объект со свойствами:
- text (string)
- completed (boolean)

Отрисовка списка происходит через функцию `render`,
которая полностью пересоздаёт DOM на основе массива данных.

## Сохранение данных
- функция `save` сохраняет массив `tasks` в localStorage через `JSON.stringify`
- функция `load` загружает данные при старте приложения и парсит их через `JSON.parse`

## Используемые технологии
- HTML
- CSS
- JavaScript (без фреймворков)
- localStorage
